# 재고 관리 시스템 구축

## 기능 정의 목록

### 제품 관리
- 제품 추가, 수정, 삭제
- 제품 코드 생성
- 제품 코드별 카테고리
  
### 재고 관리
- 제품 검색
- 제품 
  
### 입출고 관리
- 제품의 입고 및 출고 기록 관리
  
### 주문 관리
- 입출고 처리

### 계정 관리
- 거래처 등록, 수정

### 완료된 로그
- 변경된 작업 로그 출력
